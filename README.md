# Yanns Studios — Roblox Portfolio

Portfolio website for **Yanns Studios**, a Roblox game development studio founded May 2025. Built to showcase published games, live stats, and skills to collaborators and employers.

Live at: [yanns-studios.vercel.app](https://yanns-studios.vercel.app) *(update with actual URL)*

---

## What It Shows

- **5 published Roblox games** — BRAINROT TAG, Grow Bamboo For Pandas, Race Pandas, Paint or Die, Back To Rooms
- **Live stats** — visit counts and active player counts pulled from the Roblox API at request time
- **Skills** — Scripting, Game Design, Project Management, Building, UI, VFX, Animation, Rigging
- **Contact links** — Roblox profile, Discord, YouTube

---

## Tech Stack

- [Next.js 15](https://nextjs.org) (App Router, server components)
- TypeScript
- Tailwind CSS
- Roblox public API for live game stats
